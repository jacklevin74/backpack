Connect Ledger Wallet

Important: This feature requires:
  • Chrome browser with WebUSB support
  • Ledger device connected via USB
  • Solana app open on Ledger

Note: WebUSB may not be fully supported in WebView. For best results, this feature works better in a full browser.

Alternative: You can import your Ledger's public key manually using the "Import Wallet" option with your Ledger-derived public key.
